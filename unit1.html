Theory

C is a structured, procedural programming language. Every C program has:

  • Header files (like #include <stdio.h>)
  • Main function int main()
  • Statements ending with ;
  • Data types: int, float, char, double
  • Variables, constants, and operators

Example 1 – Hello World

#include <stdio.h>
int main() {
    printf("Hello, World!");
    return 0;
}

Output:

Hello, World!

Example 2 – Variables & Operators

#include <stdio.h>
int main() {
    int a = 10, b = 20;
    int sum = a + b;
    printf("Sum = %d", sum);
    return 0;
}

Output:

Sum = 30

Example 3 – Data Types

#include <stdio.h>
int main() {
    int age = 20;
    float gpa = 8.5;
    char grade = 'A';
    printf("Age: %d, GPA: %.1f, Grade: %c", age, gpa, grade);
    return 0;
}

Output:

Age: 20, GPA: 8.5, Grade: A
