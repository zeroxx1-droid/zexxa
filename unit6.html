Theory

File handling allows reading and writing data to files. Modes in fopen():

  • "r" → read
  • "w" → write (overwrites)
  • "a" → append

Example 1 – Write to File

#include <stdio.h>
int main() {
    FILE *f = fopen("test.txt", "w");
    fprintf(f, "Hello File!");
    fclose(f);
    return 0;
}

Output inside file test.txt:

Hello File!

Example 2 – Read from File

#include <stdio.h>
int main() {
    FILE *f = fopen("test.txt", "r");
    char str[50];
    fscanf(f, "%s", str);
    printf("File content: %s", str);
    fclose(f);
    return 0;
}

Output (if file has "Hello"):

File content: Hello
