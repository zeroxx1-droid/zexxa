Theory

Strings in C are arrays of characters ending with a null character \\0. Common library functions are declared in <string.h>:

  • strlen() → length of string
  • strcpy() → copy string
  • strcat() → concatenate strings
  • strcmp() → compare strings

Example 1 – String Concatenation

#include <stdio.h>
#include <string.h>
int main() {
    char s1[20] = "Hello ";
    char s2[] = "World";
    strcat(s1, s2);
    printf("%s", s1);
    return 0;
}

Output:

Hello World

Example 2 – String Length

#include <stdio.h>
#include <string.h>
int main() {
    char str[] = "Programming";
    printf("Length = %lu", strlen(str));
    return 0;
}

Output:

Length = 11
