Theory

Functions in C are blocks of reusable code. They can be:

  • User-defined
  • Library functions (like printf, scanf)
  • Recursive functions (calling themselves)

Example 1 – Function Addition

#include <stdio.h>
int add(int x, int y) {
    return x + y;
}
int main() {
    printf("Sum = %d", add(3,4));
    return 0;
}

Output:

Sum = 7

Example 2 – Recursion (Factorial)

#include <stdio.h>
int fact(int n) {
    if(n == 0) return 1;
    return n * fact(n-1);
}
int main() {
    printf("Factorial = %d", fact(5));
    return 0;
}

Output:

Factorial = 120
