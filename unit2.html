Theory

Control structures allow decision-making and looping in C. Common ones are:

  • If, if-else, nested if
  • Switch-case
  • Loops: for, while, do-while

Example 1 – Even or Odd

#include <stdio.h>
int main() {
    int num = 7;
    if(num % 2 == 0)
        printf("Even");
    else
        printf("Odd");
    return 0;
}

Output:

Odd

Example 2 – Multiplication Table

#include <stdio.h>
int main() {
    int i;
    for(i=1; i<=5; i++) {
        printf("2 x %d = %d\n", i, 2*i);
    }
    return 0;
}

Output:

2 x 1 = 2
2 x 2 = 4
2 x 3 = 6
2 x 4 = 8
2 x 5 = 10
