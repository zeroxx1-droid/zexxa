Theory

Structures allow grouping different data types under one name. Pointers are variables that store the memory address of another variable.

  • * → dereference operator
  • & → address operator
  • -> → access structure members through pointer

Example 1 – Structure with Pointer

#include <stdio.h>
struct Student {
    int id;
    char name[20];
};
int main() {
    struct Student s1 = {1, "Amit"};
    struct Student *ptr = &s1;
    printf("ID: %d, Name: %s", ptr->id, ptr->name);
    return 0;
}

Output:

ID: 1, Name: Amit

Example 2 – Pointer Basics

#include <stdio.h>
int main() {
    int x = 50;
    int *p = &x;
    printf("Value of x = %d\n", x);
    printf("Address of x = %p\n", p);
    printf("Value via pointer = %d\n", *p);
    return 0;
}

Output (example):

Value of x = 50
Address of x = 0x7ffca0...
Value via pointer = 50
